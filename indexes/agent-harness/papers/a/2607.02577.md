# Benchmarking the Benchmarks: A Validity Audit of Tool-Calling Evaluation

- ArXiv: [2607.02577](http://arxiv.org/abs/2607.02577)
- Tier: **A**
- Indexed: 2026-07-13T17:32:31+08:00

## 核心介绍

工具调用基准分数常被当作排行榜「事实」，却很少验证**评测器本身**是否可靠。本文对 BFCL v4、τ2-Bench、LiveMCPBench、MCP-Atlas 等主流工具调用基准族做系统有效性与可重复性审计。

### 关键发现（摘要口径）

- 496 个专家审查任务中，**92** 处评测器与人评分歧（约 **18.5%**）
- 确定性基准常见：脆弱匹配、轨迹锁定、错误 ground truth、子串通信失败、奖励错位
- LLM-as-judge 基准常见：评分标准漂移、幻觉完成、只评答案、**运行间方差大**
- LiveMCPBench 同设定重复 23 次，分数 **57.9%–76.8%**（差约 18.9pp），足以改变排行榜结论

### 附带产物

统一失败分类法、跟踪级审计工件、更正组件；并提出 **Tool-Veritas** 与开源 **Harness Lab**（执行、跟踪检查、重复运行比较、评测器调试）。

### 为何进 A 档

直接挑战「分数 = 能力」的默认假设，证据具体到不一致率与重复方差；读任何工具调用/harness 排行榜前应先看这类审计。

### 阅读时注意

审计覆盖的是所选基准与任务子集；不自动否定所有工具调用评测，但要求报告时分离「工具调用 / 任务完成 / 结果验证」。
