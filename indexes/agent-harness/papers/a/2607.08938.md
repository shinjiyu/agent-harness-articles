# Better Harnesses, Smaller Models: Building 90% Cheaper Agents via Automated Harness Adaptation

- ArXiv: [2607.08938](http://arxiv.org/abs/2607.08938)
- Tier: **A**
- Indexed: 2026-07-13T17:32:31+08:00

## 核心介绍

前沿 LLM 代理成本高，SLM 更便宜但直接套用为前沿模型设计的工具往往掉点。本文主张：对许多**日常业务任务**，难度有大量跨实例共享部分，可把指令、工具与编排循环从模型「抬升」到 harness；再由**元代理从故障轨迹自动发现适配**，使 SLM 以显著更低成本逼近 LLM 表现。

### 关键设计

建立故障模式 → 适配策略的映射，并实现从失败轨迹中搜索有效适配的 harness 优化器。在 **7 个面向业务的代理任务 × 3 个 SLM 家族**上评估。

### 主要结果（摘要口径）

- 21 个任务–SLM 对中 **16** 对性能显著提升
- **7** 对缩小 SLM–LLM 差距
- 最佳 SLM 代理以约 **4% 成本**恢复约 **89.7%** 的 LLM 性能
- 适配更适合**流程重复性强**、且 SLM 具备足够基础能力的任务

### 为何进 A 档

把「小模型 + 自动适配 harness」做成可操作框架，并给出清晰的成本–性能数字与适用边界，对落地选型有直接指导价值。

### 阅读时注意

「便宜 90%」强依赖任务是否偏重复业务流；开放式、高新颖性任务上收益可能下降。需核对基线 harness 与 LLM 对照是否公平。
