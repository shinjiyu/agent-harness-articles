# 9. 具身、VLA 与多模态 Harness

在机器人与多模态场景，harness 往往表现为：冻结的感知/运动策略 + 上层规划/验证/记忆编排。

## 典型结构

- **冻结 VLA / 技能检查点** 负责局部接触丰富相位
- **Planner / 验证器** 负责语义重接地、重试、重规划
- **记忆** 外化情景视觉或技能状态，避免共享窗口爆炸
- **技能交接** 是长程失败高发点：单项技能成功 ≠ 链式可启动

这些工作与文本 Agent harness 同构：都在优化「模型周围的控制面」，只是工具变成了原语库与仿真器。

### 关键要点

- 组合鲁棒性要在「脏的链式终态」上测，不能只测干净技能边界
- 结构化记忆 + 模块化决策，常比单纯堆参数更可扩展
